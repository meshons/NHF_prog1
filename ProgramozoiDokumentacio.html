NHF programozói dokumentáció - NO047V Stork Gábor

  • Fordítás és szükségletei

    A fordítás gcc-vel windows-on történhet make release, linux-on make linux paranccsal.

    A program SDL2-t és SDL2_ttf-et használ!

    Az operációs rendszernek támogatnia kell a phtread-ot.

  • Fájlok (src mappa)

    • charset.c/charset.h

      A karakterkódolásokhoz szükséges függvényeket tartalmazza, továbbá összehasonlító és átalakító függvényeket.

      • Függvények

        • void ascii_2_unicode( char * ascii, unsigned short * uni)
        • void ascii_2_unicode_a( char * ascii, atomic_ushort * uni)
        • void utf8_2_unicode_a( char * utf8, atomic_ushort * uni)
        • void latin2_2_unicode_a( unsigned char * latin2, atomic_ushort * uni)
        • void win1250_2_unicode_a( unsigned char * win1250, atomic_ushort * uni)
        • void win1252_2_unicode_a( unsigned char * win1252, atomic_ushort * uni)

          • Szimpla karakterátkódolók, mindig az első paraméterben lévő tömbből a második paraméterben lévő tömbbe.

        • void unicode_2_unicode_a( unsigned short * uni_nona, atomic_ushort * uni)
        • void char_2_char_a( char short * c, atomic_char * ca)

          • Normál tömbböket alakít át atomic tömbökké.

        • bool cmp_a_ushort( atomic_ushort * s1, atomic_ushort * s2)

          Összehasonlít két atomic_ushort tömböt és true értéket ad vissza, ha megegyezik, egyébként false

        • void digit_2_uni( int d, unsigned short * uni)

          Egész számot alakít át unicode-á max 9999-ig, min -9999-től.

    • file.c/file.h

      A rajtlista és eredményfájlból beolvasáshoz szükséges dolgokat tartalmazza.

      • Globális változók

        • extern bool error

          Hibakezeléshez változó, a main.c-ben van.

      • Struktúrák

        • struct thread_arg
          • char filename[namesize] - a beolvasandó fájl neve
          • R_list type - a beolvasás eredmény vagy rajtlista
          • Category ** list - a kategóriák listájának címe
          • int refreshrate - hány mp-enként vizsgálja a fájlt
          • R_format format - a fájl típusa, pl.: txt
          • R_charset charset - a fájl karakterkódolása pl.: win1250

      • Függvények

        • void * readerthread( void * arg)

          A beolvasó thread, paramétere a struct thread_arg.

        • R_charset TXT_getcharset( FILE * f)

          TXT fájl első sorából megállapítja a karakterkódolást.

        • Result * TXT_line( char * line, R_charset charset, R_list listtype)

          TXT eredmény vagy rajtlista fájl egy adott sorát értelmezi és visszaadja egy malloc-olt struktúrában.

        • void TXT_read( FILE * f, Category ** list, R_list listtype)

          TXT eredmény vagy rajtlista fájlt olvas be.

        • void TXT_read( FILE * f, R_charset charset, Category ** list, R_list listtype)

          IOF xml eredmény vagy rajtlista fájlt olvas be.

        • Result * CSV_line( char * line, R_charset charset, R_list listtype)

          CSV eredmény vagy rajtlista fájl egy adott sorát értelmezi és visszaadja egy malloc-olt struktúrában.

        • void CSV_read( FILE * f, Category ** list, R_list listtype, R_charset charset)

          CSV eredmény vagy rajtlista fájlt olvas be.

    • results.c/results.h

      • Definíciók

        • #define namesize 50

          Például a nevek eltárolásához használt tömb méret. Jelenleg 50 az értéke.

      • Globális változók

        • extern bool error

          Hibakezeléshez kell.

      • Felsorolások (enum)

        • typedef enum R_format

          értékek:

          • txt
          • xml
          • csv
          • undefined_format
        • typedef enum R_charset

          értékek:

          • ascii
          • utf8
          • uni
          • win1252
          • win1250
          • latin2
          • undefined_charset
        • typedef enum R_list

          értékek:

          • result
          • start

      • Struktúrák

        • typedef struct Result
          • atomic_short pos - A futó helyezése. változó
          • atomic_ushort vorname[namesize], surname[namesize] - Keresztnév, Vezetéknév
          • atomic_ushort club[namesize] - Egyesület
          • atomic_char category[namesize] - Kategória
          • atomic_int t - Futó ideje mp-ben
          • atomic_int starttime - A futó rajtideje mp-ben
          • time_t updated - Az utolsó frissitésének időpontja
          • struct Result *next - A listában a következő eredményre mutat
        • typedef struct Category
          • char name[namesize] - A kategória neve
          • atomic_int in, all - A beérkezett futók és az összes futó
          • Result *res - A kategóriában az első eredményre mutat
          • struct Category *next - A listában a következő kategóriára mutat

      • Függvények

        • void R_result_add( Category * list, Result * r, R_list listtype)

          A list kategória res-jének a végéhez fűzi az r eredményt, függően attól hogy eredmény vagy rajtlista típusú.

        • Result * R_result_new( Category * list, R_list listtype)

          A list kategória res-jének a végéhez fűz egy malloc-olt Result-ot és ennek a címét adja vissza.

        • void R_result_delete( Category * list, Result * del)

          A list kategóriából kitörli az adott del eredményt.

        • Result * R_result_search( Category * list, Result * rel)

          A list kategóriában megkeresi az adott rel eredményt.

        • void R_result_sort( Category * list)

          A list kategóriát rendezi pos alapján növekvő sorrendbe-

        • void insertionSort( Result ** head_ref)

          Segéfüggvény R_result_sort -hoz. Beszúrásos rendezés pos alapján.

        • void sortedInsert( Result ** head_ref, Result * new_node)

          Segéfüggvény R_result_sort -hoz. Rendezetten hozzáfűzi head_ref-hez new_node-t.

        • Result * R_new()

          malloc-ol egy Result -t és beállítja az értékeit alapértelmezettre, majd visszaadja a címét.

        • void deleteAFK( Category * list)

          A list kategóriából kitörli azokat az eredményeket, amik nem szerepeltek 6 percig, illetve azokat rajtlistából importáltakat, amiknek még csak rajtideje van és egy órája nem frissültek.

        • Category * R_category_new( Category ** list)

          A list kategória lista végére létrehoz egy új kategóriát és ennek a címét adja vissza. (malloc)

        • Category * R_category_find( Category * list, char * name)

          A list kategória listában megkeresi a name-el megegyező nevűt.

        • Category * R_category_find_a( Category * list, atomic_char * name)

          Ugyanaz mint az előző csak atomic_char -al;

        • void R_category_deleteresults( Category * list)

          Törli a listában lévő összes eredményt.

        • void R_category_delete( Category ** list, Category * del)

          Törli a listából az adott kategóriát és az eredményeit. Egyelőre hibás, az elsőt nem tudja törölni.

        • void R_category_deleteall( Category * list)

          Törli a listából az összes kategóriát és eredményt.

        • char * R_charset_ascii( R_charset charset)

          Visszaadja ascii karaktertömbben a charset nevét.

        • char * R_format_ascii( R_format format)

          Visszaadja ascii karaktertömbben a format nevét.

        • R_charset R_ascii_charset( char * str)

          A charset nevéből visszaadja az értékét.

        • R_format R_ascii_format( char * str)

          A format nevéből visszaadja az értékét.

        • char * R_load_cats( char * filename, int * size)

          Betölti a filename nevű fájlből a kategóriákat egy malloc-olt stringbe. *size-ba írja bele a string méretét.

        • int pow_dec( int n)

          Visszatér 10 n-edik hatványával.

    • text.c/text.h

      A ttf betűtípus betöltésével és kezelésével kapcsolatos függvényeket lát el.

      A kategóriák és eredmények képernyőn megjelenítéséhez tartalmaz függvényeket.

      • definíciók

        • #define max_glyph 319

          A Unicode karakterek Glyph -jeinek indexelésénél van szerepe.

      • Felsorolások

        • typedef enum Align

          értékek:

          • LEFT
          • RIGHT

          A betűk kiírásánal honnan írja ki.

      • Struktúrák

        • typedef struct Glyph
          • SDL_Texture * texture - a betű képe
          • SDL_Rect * rect - a betű méretei

      • Függvények

        • unsigned short getGlyphId( unsigned short i)

          Az Unicode karakterkódból előállítja az indexét.

        • unsigned short getGlyphCH( unsigned short i)

          Az indexéből előállítja az Unicode karakterét.

        • void loadGlyphs( Glyph * g, TTF_Font * font, SDL_Color color, SDL_Renderer * renderer)

          Betölti a g-be a font-ban lévő karaktereket.

        • void destroyGlyphs( Glyph * g)

          Törli a g-ben lévő karaktereket.

        • void text( unsigned short * uni, SDL_Renderer * renderer, Glyph * g, int x, int y, Align a)

          Megjeleníti az uni-ban lévő betűket x y pozícióban, g betűkkel. a alapján balra vagy jobbra rendezve.

        • void texta( atomic_ushort * uni, SDL_Renderer * renderer, Glyph * g, int x, int y, Align a)

          Mint az előző, csak atomic_ushort -al.

        • void background( SDL_Renderer * renderer, int x, int y, int w, int h, SDL_Color * c)

          Egy sima téglalapot rajzol c színnel. x y helyre, w h kiterjedéssel, c színnel.

        • void render_result( SDL_Renderer * renderer, int x, int y, int w, int h, SDL_Color * back, Result * res, int * first, int * nulltime)

          Megjeleníti egysorban a res eredményt back háttérrel. x y hely,re, w h kiterjedéssel, back a háttér színe. first a kategóriában az elsőnek az eredmény ideje, nulltime a verseny nullideje, amikor rajt van.

        • void render_category( SDL_Renderer * renderer, int x, int y, int w, int *size, Category * c int * nulltime)

          Megjeleníti a c kategória eredményeit. *size-ba írja be az összemagasságát. x y helyre, w szélességgel. nulltime a verseny nullideje.
